background-color specifies color of background of an element and set it for any HTML element

opacity ranges from 0.0-1.0 and specifies opacity/transparency of element, the lower, the more transparent,
all child elements inherit same opacity

Having background-image covers entire background of element, it can be used for p-tags and other elements

Having background-repeat: repeat-x in any element will prevent background img from repeating in horizontal direction

Having background-repeat: repeat-y in any element will prevent background img from repeating in vertical direction

Having background-repeat: no-repeat in any element will prevent background img from repeating

The background-position: right top specifies the position of background img

The background-attachment: fixed specifies whether background img should be scrolled or fixed with rest of page
fixed will scroll with page, scroll will not scroll with page

background: #ffffff url("img_tree.png") no-repeat right top;

background-color, background-image, background-repeat, background-attachment, background-position can be used in single line
Must occur in this order, but some entries can be missing
