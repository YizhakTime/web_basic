Outline is a line drawn outside element's border. Content is inside border which is inside outline

Outline is NOT part of element's dimensions and use outline-style property to change type of outline

outline-width specifies width of outline and can either be:

  • thin (1px)
  • medium (3px)
  • thick (5px)
  • any specific size

outline-color sets color of outline via name, hex, rgb, hsl, or invert

outline has a shorthand style to combine all of these properties (color, width, style (required)), order does not matter
p.ex4 {outline: thick ridge pink;}

outline offset adds space between edge of element and outline and it is transparent
