height/width sets height and width of element

20x70 div

height/width have these properties: auto (default, browser calculates height/width), length, % (% of containing block),
initial (sets height/width to default value), inherit (height/width will be inherited from parent element)

max-width specifies max width of element, issue w/div is when browser window is smaller than width of element (500px),
browsers adds horizontal scrollbar, use max-width to handle smaller windows
if you use width and max-width and width value > max-width, max-width will be used
