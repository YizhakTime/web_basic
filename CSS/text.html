text-decoration sets decoration to text: overline sets line over text, line-through strikethroughs text, underline underlines text, and overline underline sets a line under and over text

text-decoration-color sets color of decoration

text-decoration-style sets style of decoration line with solid, double, dotted, dashed, underline, wavy,

text-decoration-thickness specifies thickness of decoration line

text-decoration property is shorthand for text-decoration-line (required), text-decoration-color, text-decoration-style, text-decoration-thickness

text-decoration: none removes underline from links, which are the default
