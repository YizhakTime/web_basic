Have desired font as first font and have generic font as last font for browser to use

There's no 100% web-safe font, so include list of web-safe fonts in font-family to be safe

Include arial, verdana, TNR, Tahoma, Georgia, ...

Other font fallbacks are serif, sans-serif, monospace, cursive, and fantasy

font-style allows for italics, font-weight specifies weight of font, font-variant allows small-caps font -> all uppercase but OG uppercase are bigger

font-size sets size of text, can be absolute (set size and cannot be modified) or relative (size relative to elements/changes text ),

use vw or viewport-width to see how text changes following size of the browser

To use google fonts, use stylesheet in link tag along with api_key

To use multiple Google fonts, use the (|) symbol in the href tag

Use (|) to attach multiple effects, effects start with font-effect and then name of effect

The font can be shortened to font-style, font-variant, font-weight, font-size/line-height, and font-family. font-size and font-family are required
