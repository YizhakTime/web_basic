• Simple selectors: select elements based off id, name, class
  • The p brackets is a simple selector
  • id selector uses id for attributes of an element, select one UNIQUE element
    id cannot start with number, to use id, use # followed by id
  • class selector selects HTML element with specific class attribute, must have period followed by class name
  • Combinator selectors: select elements based off specific relationship between them
  • pseudo-class selectors: select elements based on a certain state
  • pseudo-elements-selectors: select and style a part of an element
  • attribute selectors: select elements based on an attribute or attribute value
