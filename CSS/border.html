border-style defines type of border like dotted, dashed, solid, double, but border-style has to be defined and border-width specifies width

border width can have 4 values: top, right, bottom, left border

border-color specifies color of border, can be through name, rgb, hsl, hsla, hex, transparent
Since border has 4 sides, you can give border multiple colors for all of its sides

You can specify which borders have different styles

You could also use this format: border-style: dotted solid double dashed
order is as follow for 4 sides: top, right, bottom, left

If border-style has 3 borders: border-style: dotted solid double; top is dotted, right and left are solid, and bottom is double

If border-style has 2 borders: border-style: dotted solid; top/bottom are dotted, left/right are solid, this works with border-color and border width

border: 5px solid red; follows border attributes as border-width, border-style, border-color
This does it for just one side: border-left: 6px solid red;

border-radius add rounded borders to element

Margin is used to create space around elements, outside defined borders, you can specify the margin for each side of the border
Margin can have these values: auto (browser calculates margin), length, % margin percentage of width of containing element
inherit specifies margin that should be inherited from parent element

These are all of the directions for margin: margin-top: 100px, margin-bottom: 100px, margin-right: 150px, margin-left: 80px;

margin: 25px 50px 75px 100px is shorthand for specifiying margin-top, margin-bottom, etc...

margin: 25px 50px 75px is 3 sided, top is 25px, right/left is 50px, and bottom is 75px

margin: 25px 50px; top/bottom is 25px and left/right is 50px

shorthand value for all 4 margins

Auto horizontal centers element within container between left and right margins

Child element can inherit from parent element
