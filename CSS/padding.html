Padding property generates space around element's content inside any defined border, there are elements for ALL 4 sides
padding-top, padding-left, padding-right, padding-bottom
Attributes are length, % (padding in % of width of containing element),
inherit (which padding to be inherited from parent element), negative values are NOT allowed

Has 4 sides with different paddings

All of these properties can be simplified into one property:
padding: 25px 50px 75px 100px, top has 25px, right has 50px, bottom has 75px, left has 100px
padding: 25px 50px 75px, top has 25px, right/left are 50px, and bottom is 75px
With 2 values top/bottom has 1st value, right/left have 2nd value, with only ONE value, ALL paddings have the same value

width usually specifies width of element's content area,
so if element has specified width, padding will be added to total width of element

First div
300px + 25px (left) + 25px (right) = 350px
