Box model refers to design and layout, box that wraps around every HTML element.
Consists of content, padding, borders, and margins

box model allows us to add border around elements, define space between elements

Example of box model

To calculate total width/height, you need to include borders, padding

320px (width of content area) + 20px (left padding + right padding) + 10px (left border + right border) = 350px (total width) 50px (height of content area) + 20px (top padding + bottom padding) + 10px (top border + bottom border) = 80px (total height)

The total width of an element should be calculated like this: Total element width = width + left padding + right padding + left border + right border The total height of an element should be calculated like this: Total element height = height + top padding + bottom padding + top border + bottom border
